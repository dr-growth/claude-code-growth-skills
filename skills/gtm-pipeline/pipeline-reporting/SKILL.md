# Pipeline Reporting

Generate campaign ROI reports, pipeline metrics, and performance dashboards. Pulls from campaign-feedback data, CRM pipeline, and analytics to give a complete picture of marketing performance.

## When to Use

- Weekly pipeline review preparation
- Campaign post-mortem analysis
- Monthly/quarterly marketing performance reporting
- When you need to show ROI to leadership

## Inputs

1. **Time period** — week, month, quarter, or custom range
2. **Report type** — pipeline, campaign ROI, channel attribution, or executive summary
3. **Data sources** — campaign-feedback output, CRM data, GA4, Semrush

## Report Types

### Pipeline Report
- MQLs generated by source/campaign
- Pipeline value by stage
- Conversion rates between stages
- Velocity metrics (days in stage)

### Campaign ROI
- Cost per campaign (ad spend, tool costs, time)
- Leads generated per campaign
- Pipeline attributed to each campaign
- CAC by campaign and channel

### Channel Attribution
- First-touch vs multi-touch attribution
- Channel contribution to pipeline
- Content performance by topic/format
- Organic vs paid vs outbound split

### Executive Summary
- Top-line metrics: MQLs, pipeline, revenue influence
- Month-over-month trends
- Top performing campaigns
- Recommendations for next period

## Output

Markdown report with embedded data tables. Optionally generates HTML dashboard for visual presentation.

## Dependencies

- campaign-feedback skill output
- tracking-analytics data (GA4)
- CRM pipeline data (when available)
